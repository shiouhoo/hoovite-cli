## 快速使用

首先将项目代码克隆到本地，建议使用 `npm`,或 `yarn` 安装,如果您的网络环境不好，建议使用相关镜像服务

```js
// https
git clone https://gitee.com/shiouhoo/hooadmin.git

// ssh
git clone git@gitee.com:shiouhoo/hooadmin.git
```

安装依赖

```js
// npm
npm install

// yarn
yarn
```

本地启动服务器来开发你的网站，项目默认运行在 8080 端口

```js
// npm
npm run serve

// yarn
yarn serve
```
